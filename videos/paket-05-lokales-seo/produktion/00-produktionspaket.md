## 3 — Topic Intelligence Report
- **Einordnung:** Erklär-/Konkurrenz-Video — liefert das mentale Modell hinter
  allen anderen Paketen (Relevanz, Entfernung, Bekanntheit).
- **Zielzuschauer:** Betrieb, der frustriert ist, weil ein „schlechterer"
  Konkurrent höher rankt.
- **Hook:** Konkurrenz-Neid + „selten Zufall" → verspricht das Warum + den
  Aufholplan.
- **Struktur:** Hook → Signal 1 Relevanz (+ Konkurrenzvergleich) → Signal 2
  Entfernung (nicht änderbar) → Signal 3 Bekanntheit → NAP-Check (unterschätzt)
  → Aufholhebel-Fazit → Abo.
- **Retention:** „3 Signale" Rahmen, Konkurrenzvergleich, überraschender
  NAP-Punkt, klarer erster Schritt.
- **Länge:** ~4–5 Min. · **Emotion:** von Frust → zu Klarheit & Plan.
