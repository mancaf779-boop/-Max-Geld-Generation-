## 3 — Topic Intelligence Report

- **Nischen-Einordnung:** „Fehler aufdecken"-Format — starker Hook-Typ, weil es
  Angst/Neugier triggert („mache ich einen davon?"). Baut auf Paket 01 auf.
- **Zielzuschauer:** Betrieb hat schon ein Profil, aber es kommen keine
  Anfragen — sucht die Ursache.
- **Hook-Strategie:** Problem-Frage + Open Loop („Fehler Nummer 3 kostet am
  meisten Geld") → hält bis zur Mitte.
- **Struktur:** Hook → Fehler 1 Kategorie → Fehler 2 leeres Profil → Fehler 3
  Fotos (der teuerste) → Fehler 4 Bewertungen → Fehler 5 kein Buchungsweg →
  Abhaken-CTA → Abo.
- **Retention:** nummerierte Liste (klarer Fortschritt), Open Loop auf Fehler 3,
  „gratis Ranking", Sofort-Abhaken.
- **Länge:** ~4–5 Min. · **Emotion:** von Sorge → zu Erleichterung/Kontrolle.
